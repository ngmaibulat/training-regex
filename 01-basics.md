### CLI Basics

```bash
pwd
mkdir tmp
cd tmp
pwd

echo "hello"
echo "hello" > file.txt
echo "world" >> file.txt

ls -l
cat file.txt

cd ..
rm -rf tmp
```

### List of commands

-   `pwd` - print working directory
-   `mkdir` - make directory
-   `cd` - change directory
-   `echo` - print to stdout
-   `ls` - list directory contents
-   `cat` - concatenate and print files
-   `rm` - remove files or directories
